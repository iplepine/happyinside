# 행복한 순간 기록하기와 관련된 피쳐

## 🎯 주요 기능 (MVP)

- ✍️ **행복한 순간 작성 화면**
  - 행복한 순간 작성
  - 필수값, 필수가 아닌 값 구분
  - 값 입력 시 다음 값 바로 입력 가능하도록 처리
  - 저장 CTA 버튼은 하단에 항상 노출

- 📋 **기록 목록 보기 화면**
  - 타임라인 또는 카드뷰 형태
  - 날짜/태그/강도별 정렬 지원
  - 감정 통계 요약 화면으로 이동

- 📊 **감정 통계 요약 화면**
  - 주간/월간 기록 수
  - 자주 등장한 단어/태그
  - 감정 강도 평균, 가장 행복했던 날 등
